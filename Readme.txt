1.	Download Requirements for the program
	- Python 3.9.12
	- Anaconda
	- Tenserflow
	- Keras
	- Juypter Notebook

2.	Install the required packages
	- Tenserflow (pip install tensorflow)
	- NumPy (pip install numpy)
	- Pandas (pip install pandas)
	- Matplotlib (pip install matplotlib)
	- Ntlk (pip install nltk)
	- keras (pip install keras)

3.	Extract the project files in a New Volume under a specific name (eg. myproject)
4.	Configure Anaconda (Python 32) to the implementation folder
5.	Initiate Jupyter Notebook in Anaconda Terminal
6.	Once the Jupyter Notebook is connected to the local host, we can get access to the implementation part on the local server
7.	Open the main code and clear existing outputs / errors by choosing Kernal -> Restart and Clear all Output
8.	Once all the outputs got cleared, run the entire code by choosing Cell -> Run all
9.	Finally, the desired output will be attained on the specific steps.

Note : Incase of any errors, try configuring anaconda prompt properly or try reinstalling the required packages
